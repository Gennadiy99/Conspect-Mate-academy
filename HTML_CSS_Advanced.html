Налаштування середовища

Для виконання завдань цього та наступних модулів необхідно встановити деякі інструменти. Правильне налаштування середовища допоможе тобі ефективно навчатися та працювати над проєктами.

Що таке Node.js?

Node.js — це вільне та відкрите середовище виконання JavaScript, яке побудоване на двигуні V8 JavaScript Engine компанії Google Chrome. Це означає, що ти можеш створювати вебдодатки з використанням однієї мови на обох сторонах — клієнтській та серверній.

Чому це потрібно?

Node.js надає безліч переваг:

  1. Швидкість: завдяки асинхронній обробці запитів, Node.js може обробляти багато запитів одночасно.
  2. Універсальність: виконання JavaScript не тільки в браузері відкриває розробникам можливість створення серверних додатків без необхідності вивчення нової мови програмування.
  3. Велика спільнота: завдяки великій спільноті ти завжди можеш знайти бібліотеки та ресурси, які спростять твою роботу.

    4. У цьому модулі ти не будеш занурюватись у використання Node.JS. Він знадобиться нам для підключення і роботи зі сторонніми бібліотеками.

Встановлення Node.js 20

Щоб встановити Node.js 20 собі на комп'ютер, виконай наступні кроки:

  1. Завантаж 64-розрядний інсталятор для твоєї операційної системи з цієї сторінки.
  2. Запусти його та дійди до кінця залишаючи всі налаштування за замовчуванням.
  3. Відкрий термінал (GitBash для Windows або ZSH для macOS).
  4. У терміналі виконай команду node -v. Ти маєш побачити версію v20.xx.x.

Встанови останні версії розширень VSCode

Використання розширень у VSCode допоможе підвищити продуктивність і зручність роботи. Ось кілька рекомендованих розширень:

  • ESLint: допомагає підтримувати код у чистоті та відповідності стилю.
  • EditorConfig: створює єдині правила форматування для різних редакторів.
  • Stylelint: перевіряє CSS на помилки та недоліки.
  • LintHTML: перевіряє HTML-код на помилки.
  • Prettier: автоматично форматує код, роблячи його більш читабельним.

NPM

NPM (Node Package Manager) — це менеджер пакетів для Node.js, який дозволяє управляти бібліотеками та інструментами, які ти будеш використовувати у своїх проєктах.

Чому це потрібно?

  • Управління залежностями: з NPM ти можеш легко встановлювати, оновлювати та видаляти необхідні модулі. Це можуть бути як CSS-бібліотеки, так і інструменти для збірки проєкту.
  • Широкий вибір бібліотек: у NPM доступні тисячі готових бібліотек, які можуть прискорити твою розробку.

Як використовувати NPM?

  1. У терміналі введи команду npm init, щоб створити файл package.json. Цей файл міститиме інформацію про твій проєкт і його залежності.
  2. Щоб встановити новий пакет, використай команду npm install <назва пакета>. Наприклад, npm install express для встановлення бібліотеки Express.
  3. Щоб встановлений пакет також додався до файлу package.json, при виклику npm install треба передати параметр –save. Наприклад, npm install express –save.
  4. Після налаштування Node.js та NPM, ти можеш використовувати різноманітні інструменти, які спростять процес розробки. Ось кілька з них:
  • Parcel — інструмент для пакування вебдодатків. Він пропонує швидку роботу з використанням багатоядерної обробки та не вимагає налаштувань.
  • Bootstrap/Tailwind — популярні CSS-фреймворки, що можуть спростити процес створення вебінтерфейсів.
  • React/Angular — JavaScript-фреймворки для створення динамічних односторінкових вебдодатків.

Знайомство з Figma

У процесі навчання і в подальшій роботі тобі потрібно буде працювати з Figma, оскільки цей інструмент є популярним серед дизайнерів. Опанувавши його, ти зможеш легко отримувати доступ до макетів, переглядати специфікації (кольори, шрифти, розміри) та відтворювати дизайн у коді.

Тому перш ніж перейти до навчання та перших завдань, потрібно зареєструватися в Figma. Для цього:

  1. Перейди на офіційний сайт Figma.
  2. Натисни на кнопку Get started for free на сторінці або в правому верхньому кутку.
  3. Введи свою електронну пошту та придумай пароль, або скористайся реєстрацією через Google. Натисни на кнопку Create account.
  4. Перевір свою електронну скриньку, знайди лист від Figma з темою Verify your email address і натисни на кнопку Verify email.
  5. Заповни дані про себе і дай відповідь на декілька запитань стосовно планів на використання Figma (ти можеш обрати будь-які варіанти відповіді на них на свій вибір).
  6. Реєстрація майже завершена, залишилося обрати план — для навчання тобі буде достатньо Starter.

Тепер ти можеш почати працювати з Figma! А в наступному відео ти дізнаєшся, як саме використовувати Figma для виконання завдань.
